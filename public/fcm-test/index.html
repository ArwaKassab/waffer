🚀 اختبار إشعارات Firebase (Web Push)

هذه الصفحة تقوم بـ:
1) تحميل إعدادات Firebase من /firebase-config.js.
2) تسجيل Service Worker: /firebase-messaging-sw.js.
3) طلب إذن الإشعارات من المتصفح.
4) جلب FCM Token عبر VAPID وإظهاره وإرساله لـ API الخلفية.

ملاحظة: زبّطي روابط الـAPI بما يناسب مشروعك (داخل الكود أدناه ستجدي نقاط التعويض).
